# Portfolio Website

This repository contains the source code for my personal portfolio website.

The purpose of this project is to design and develop a clean, responsive, and accessible portfolio that showcases my skills, projects, and learning journey as an Information Technology student and aspiring Full Stack Developer.

---

## Project Goals
- Build a fully responsive personal portfolio website
- Apply semantic HTML and modern CSS architecture
- Practice clean UI layout and accessibility best practices
- Gradually enhance the project with real-world features

---

## Tech Stack
- HTML5
- CSS3 (Modular architecture: variables, base, layout, components)
- JavaScript (for interactivity)
- Git & GitHub for version control
